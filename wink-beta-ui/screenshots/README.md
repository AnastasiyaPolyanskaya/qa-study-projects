# Скриншоты - Wink Beta (UI)

В данном разделе размещены скриншоты, полученные в ходе UI-тестирования главной страницы сайта **Wink Beta**.

Скриншоты используются как визуальные доказательства обнаруженных дефектов и подтверждают фактическое состояние интерфейса на момент тестирования.

---

## Содержание
- Скриншоты главной страницы сайта Wink Beta
- Изображения с воспроизведением UI-дефекта (обрезание текста)
- Визуальное сравнение ожидаемого и фактического отображения элементов

---

## Назначение скриншотов
Скриншоты применяются для:
- подтверждения найденных багов;
- иллюстрации фактического результата в баг-репортах;
- демонстрации навыков визуального анализа интерфейса.

---

## Связанные артефакты
- Описание дефектов находится в каталоге `bug-reports/`
- Тест-кейсы, по которым выполнялась проверка, находятся в каталоге `test-cases/`

---

## Примечание
Материалы представлены в учебных целях и предназначены для демонстрации навыков ручного UI-тестирования.
