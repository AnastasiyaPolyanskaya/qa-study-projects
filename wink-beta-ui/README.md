# Проект Wink Beta - Тестирование главной страницы (UI)

## Описание  
Исследовательское тестирование пользовательского интерфейса сайта Wink Beta-тестирование.  
Основное внимание уделено корректности отображения элементов главной страницы и адаптивности блоков при просмотре на мобильных устройствах.

## Окружение  
- Устройство: iPhone 14 Pro Max  
- ОС: iOS 18.6.2  
- Браузер: Яндекс.Браузер  
- Тип тестирования: UI, исследовательское  
- Применимость: мобильная версия сайта  

## Проверенные сценарии  
1. Проверка корректности отображения текста под блоком «Шаги» на главной странице  
2. Проверка адаптивности секции «Как принять участие?» при скролле вниз  

## Найденные дефекты  
1. **Некорректное отображение текста под блоком «Шаги» на главной странице Wink Beta**  
   - Ожидаемый результат: текст под блоком «Шаги» отображается полностью  
   - Фактический результат: часть текста обрезана, окончание отображается не полностью  
   - Серьёзность: Minor  
   - Уровень: Medium  
   - Комментарий: текстовый блок не адаптируется под разрешение экрана, вероятно нарушены отступы вёрстки  

## Прикреплённые файлы  
- `Тест-кейсы_wink.xlsx` — тест-кейсы по проверке главной страницы  
- `Баг-репорты(wink).docx` — баг-репорты по найденным дефектам  
- `screenshots/` — скриншоты с воспроизведением дефекта  

## Итог  
Проведено исследовательское тестирование интерфейса сайта Wink Beta.  
Найден дефект, связанный с некорректным отображением текста на мобильной версии.  
Рекомендовано проверить адаптивность секции для устройств с разными разрешениями экрана.
