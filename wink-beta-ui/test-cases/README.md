# Тест-кейсы - Wink Beta (UI)

В данном разделе размещены тест-кейсы, подготовленные для проверки пользовательского интерфейса (UI) главной страницы сайта **Wink Beta**.

Тестирование выполнено в рамках учебного проекта и направлено на выявление визуальных и адаптивных проблем мобильной версии сайта.

---

## Объект тестирования
- Главная страница сайта Wink Beta
- Основные текстовые и навигационные блоки интерфейса

---

## Тип тестирования
- UI-тестирование
- Исследовательское тестирование (Exploratory)

---

## Проверяемые сценарии
- Корректность отображения текста под блоком «Шаги»
- Адаптация текстовых блоков под мобильное разрешение
- Поведение интерфейса при скролле страницы
- Отсутствие обрезаний текста и визуальных артефактов

---

## Формат тест-кейсов
Тест-кейсы оформлены в табличном виде и включают:
- ID тест-кейса
- Описание шага
- Предусловия
- Шаги выполнения
- Ожидаемый результат
- Фактический результат
- Статус (Passed / Failed)

---

## Связанные артефакты
- Баг-репорты по найденным дефектам расположены в каталоге `bug-reports/`
- Скриншоты воспроизведения дефектов — в каталоге `screenshots/`

---

## Примечание
Тест-кейсы предназначены для демонстрации навыков ручного тестирования UI и не являются коммерческой документацией.
