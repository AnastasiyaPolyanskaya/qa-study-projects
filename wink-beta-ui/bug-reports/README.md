# Баг-репорты - Wink Beta (UI)

В данном разделе собраны баг-репорты, обнаруженные в ходе исследовательского UI-тестирования главной страницы сайта **Wink Beta**.

Дефекты зафиксированы в рамках учебного проекта и оформлены по стандартной структуре баг-репорта с указанием шагов воспроизведения, ожидаемого и фактического результата.

---

## Объект тестирования
- Главная страница сайта Wink Beta
- Текстовые и информационные блоки интерфейса

---

## Тип найденных дефектов
- UI-дефекты
- Ошибки адаптивности
- Ошибки отображения текста

---

## Краткий обзор дефектов

### Баг №1 — Обрезание текста под блоком «Шаги»
- **Тип:** UI / Адаптивность
- **Серьёзность:** Minor
- **Уровень:** Medium

**Описание:**  
Текст под блоком «Шаги» на главной странице отображается не полностью — часть текста обрезана на мобильном разрешении.

**Ожидаемый результат:**  
Текст отображается полностью и корректно переносится на новую строку.

**Фактический результат:**  
Нижняя часть текста скрыта, читаемость нарушена.

**Комментарий:**  
Вероятная причина — некорректные отступы или фиксированная высота текстового блока.

---

## Формат баг-репортов
Каждый баг-репорт содержит:
- ID дефекта
- Название
- Окружение
- Предусловия
- Шаги воспроизведения
- Ожидаемый результат
- Фактический результат
- Серьёзность и уровень
- Комментарий
