# Баг-репорты — Stepik.org

Раздел содержит дефекты, найденные при тестировании форм регистрации и авторизации на платформе Stepik.

## Формат баг-репортов
Каждый баг описан по структуре:

- **ID дефекта**
- **Заголовок**
- **Окружение** (браузер, ОС, устройство)
- **Предусловия**
- **Шаги воспроизведения**
- **Ожидаемый результат**
- **Фактический результат**
- **Серьёзность / приоритет**
- **Артефакты** (скриншоты, видео — если есть)

## Краткое описание найденных дефектов
- Ошибки валидации полей  
- Некорректные UI-сообщения  
- Неверная обработка неверных данных  
- Ошибки в работе кнопок и навигации  
- Несоответствия мобильной версии  
- Серверные ответы с ошибками (DevTools)

## Назначение раздела
Папка служит для демонстрации твоей способности:
- находить и фиксировать ошибки,
- оформлять понятные и воспроизводимые баг-репорты,
- работать с DevTools,
- анализировать поведение системы в разных окружениях.
