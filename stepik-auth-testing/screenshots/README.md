# Скриншоты — Stepik.org

Раздел содержит визуальные материалы, использованные в процессе тестирования форм регистрации и авторизации на платформе Stepik.

## Что содержится в этой папке
- Скриншоты ошибок, найденных во время тестирования  
- Подтверждения шагов воспроизведения дефектов  
- Скриншоты валидации полей  
- UI-несоответствия  
- Примеры сетевых запросов (если делались снимки DevTools)  
- Материалы для наглядного подтверждения баг-репортов

## Стандарты оформления
Чтобы скриншоты выглядели единообразно:

- Файлы именуются по шаблону:  
  **`<ID_дефекта>_<краткое_описание>.png`**
  - Например: `BR-012_invalid_email_validation.png`

- Разрешение скриншотов: не менее **1280×720**  
- При необходимости выделены проблемные зоны (рамки, стрелки)

## Назначение раздела
Скриншоты помогают:
- подтвердить фактическое поведение системы,  
- ускорить анализ дефекта,  
- облегчить проверку задачи ревьюеру или работодателю,  
- повысить качество баг-репортов и тест-кейсов.
