# Stepik: Тестирование форм авторизации

Учебный проект (дипломная работа).  
Проект посвящён исследовательскому и структурному тестированию форм регистрации и авторизации на платформе Stepik.

## Содержимое проекта

- **test-cases/** — тест-кейсы для позитивных и негативных сценариев  
- **checklists/** — чек-листы по функциональности  
- **bug-reports/** — баг-репорты  
- **screenshots/** — визуальные материалы (опционально)

## Статус
Проект находится в процессе переноса и структурирования.

## Цель работы
Отработка практических навыков тестирования:
- анализ требований  
- тест-дизайн  
- составление тестовой документации  
- поиск и оформление дефектов
