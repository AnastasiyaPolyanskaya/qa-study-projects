# Stepik.org — тестирование форм авторизации и регистрации

## Описание проекта
Учебный исследовательский проект по тестированию форм регистрации и авторизации на платформе Stepik.  
В рамках работы проведено ручное функциональное тестирование, UI-проверки, анализ валидации, проверка мобильной версии и анализ сетевых запросов.
