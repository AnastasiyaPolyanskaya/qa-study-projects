# Чек-листы — Stepik.org

Раздел содержит чек-листы, созданные в рамках тестирования форм регистрации и авторизации на платформе Stepik.

## Содержание
- Проверка UI-элементов
- Проверка функциональности кнопок
- Проверка валидации полей
- Проверка обработки ошибок
- Проверка мобильной версии (iOS / Android)
- Проверка нестандартных и edge-case сценариев

## Описание
Чек-листы представлены в структурированном виде и предназначены для оценки качества интерфейса, логики и поведения системы при вводе корректных и некорректных данных.
