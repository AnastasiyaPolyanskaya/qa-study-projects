# Работа с числовыми переменными (Python)

Учебный мини-проект, выполненный в рамках практики по Python.  
Задачи показывают базовые навыки работы с числами, вводом данных, арифметическими операциями и форматированием вывода.

---

## Содержимое проекта

В рамках мини-проекта отрабатываются:

- ввод чисел через `input()` и преобразование типов (`int`, `float`);
- арифметические операции: сложение, вычитание, деление, умножение, проценты;
- расчёт прибыли с учётом налогов;
- возведение в степень;
- создание простого консольного калькулятора;
- различные форматы вывода (округление, форматирование строк).

---

## Файлы проекта

| Файл | Описание |
|------|----------|
| `task_1_sum.py` | Сложение двух чисел, вводимых пользователем. |
| `task_2_profit.py` | Расчёт чистой прибыли после удержания налога. |
| `task_3_power.py` | Возведение числа в степень. |
| `task_4_percent.py` | Вычисление процента от числа. |
| `task_5_calculator.py` | Простой калькулятор из четырёх операций. |
| `task_6_output_formats.py` | Форматированный вывод чисел (округление, f-строки). |

---

## Итог

Проект демонстрирует умение работать с числовыми данными в Python, выполнять вычисления и оформлять результат в удобочитаемом виде.
