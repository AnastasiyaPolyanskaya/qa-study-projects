# Работа с числовыми переменными (Python)

Учебный мини-проект, выполненный в рамках практики Python.  
Задачи показывают базовые навыки работы с числами, вводом данных, арифметическими операциями и выводом результата.

## Содержимое проекта

- ввод чисел через `input()` и преобразование в `int`;
- арифметические операции (сложение, деление, возведение в степень);
- расчёт чистой прибыли с учётом налоговой ставки;
- варианты вывода данных с подсказками.

## Файлы проекта

| Файл | Описание |
|------|----------|
| `task_1_sum.py` | Сложение двух чисел через `input()` |
| `task_2_sum_function.py` | Сложение через встроенную функцию `sum()` |
| `task_3_profit.py` | Расчёт чистой прибыли после удержания налога |
| `task_4_power.py` | Возведение числа в степень |
| `task_5_power_short.py` | Короткое возведение в степень |
| `task_6_sum_input.py` | Ввод чисел с подсказками и вывод результата |

## Итог

Раздел демонстрирует умение работать с простыми числовыми операциями в Python и аккуратно оформлять код.
