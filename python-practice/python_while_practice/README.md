# Практика Python - цикл `while`

Учебный проект для закрепления работы с циклом `while` на базовых задачах: перебор списков, условия остановки, фильтрация и изменение элементов.

---

## Окружение
- Язык: Python 3
- Ввод данных: `input()`
- Формат ввода: элементы в одну строку через пробел
