# Python practice

Раздел с учебными мини-проектами по Python.  
Все задачи делались в рамках обучения и нужны для портфолио тестировщика: здесь видно, что я умею работать с переменными, строками, списками, циклами и функциями.

## Список мини-проектов

### 1. Проект №7.1 — Работа с числовыми переменными
**Папка:** `python_numeric_practice/`  
**Темы:** ввод чисел через `input()`, преобразование в `int`, арифметические операции, возведение в степень, расчёт чистой прибыли с учётом налога, варианты вывода с подсказками.

### 2. Проект №7.2–7.3 — Работа со строками
**Папка:** `python_string_practice/`  
**Темы:** индексация строки, срезы, шаг среза, отрицательные индексы, вывод отдельных символов, конкатенация строк, разворот строки, подсчёт и удаление символов, изменение регистра.

### 3. Проект №7.4 — Строки и f-строки
**Папка:** `python_strings_and_fstrings/`  
**Темы:** форматирование вывода через f-строки, работа со строками, расчёт возраста фильма, простые расчёты (обмен валют и др.) с красивым текстовым выводом.

### 4. Проект №7.5 / №7.7 — Ввод, вывод и функции
**Папка:** `python_functions_basics_7_4/`  
**Темы:** функции `summ()` и `hi()`, позиционные и именованные аргументы, использование глобальных переменных, ввод данных пользователя через `input()`, формирование строк, первый опыт с `return`.

### 5. Проект №7.6 — Функции и оператор `return`
**Папка:** `python_functions_return/`  
**Темы:** функции суммы, среднего арифметического, расчёта скидки, собственные реализации `max` и `min`, использование результата функции в дальнейших вычислениях, мини-калькулятор из четырёх операций.

### 6. Проект №7.8 — Списки, строки и циклы
**Папка:** `python_lists_strings_practice/`  
**Темы:** обход списка по индексам, вычисление среднего значения, фильтрация символов строки, подсчёт количества букв, создание списков в цикле, сравнение сумм двух списков.
