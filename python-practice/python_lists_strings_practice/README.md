# Работа со списками, строками и f-строками (Python)

Учебный мини-проект, выполненный в рамках практики Python.  
Проект демонстрирует навыки работы со списками, строками, поиском, сравнением и форматированным выводом через f-строки.

## Содержимое проекта

- работа с индексами списка;
- вычисление суммы, длины и среднего значения;
- фильтрация символов строки;
- сравнение двух списков и поиск общих элементов;
- подсчёт количества букв верхнего и нижнего регистра;
- нахождение минимального и максимального значения списка;
- форматирование результата через f-строки (где применимо).

## Файлы проекта

| Файл | Описание |
|------|----------|
| `task_1_list_index.py` | Работа с индексами списка: выбор по позиции, последний элемент, произвольный индекс. |
| `task_2_average_value.py` | Расчёт суммы, количества и среднего значения элементов списка. |
| `task_3_filter_letters.py` | Фильтрация строки: извлечение только букв из текста. |
| `task_4_compare_lists.py` | Сравнение двух списков и поиск общих элементов. |
| `task_5_count_upper_lower.py` | Подсчёт букв верхнего и нижнего регистра в строке. |
| `task_6_min_max_list.py` | Определение минимального и максимального значения списка. |
