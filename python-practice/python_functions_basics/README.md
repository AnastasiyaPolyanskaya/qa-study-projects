# Работа с функциями на Python

Учебный мини-проект, выполненный в рамках практики по Python.  
Задачи показывают базовые приёмы работы с функциями, параметрами, возвратом значений и форматированием вывода.

## Содержимое проекта

В рамках мини-проекта отрабатываются:

- объявление и вызов функций;
- передача аргументов в функции;
- использование нескольких параметров;
- возврат значения из функции и работа с результатом;
- формирование текстового отчёта через f-строки.

## Файлы проекта

| Файл                         | Описание |
|-----------------------------|----------|
| `task_1_sum.py`             | Простейшая функция сложения двух чисел и вывод результата. |
| `task_2_hi.py`              | Функция-приветствие, принимающая имя и выводящая сообщение. |
| `task_3_sum.py`             | Примеры вызова функции сложения с разными аргументами. |
| `task_4_hi.py`              | Варианты функции `hi()` с несколькими параметрами (имя, возраст и т.п.). |
| `task_5_string_format_report.py` | Формирование оформленного текстового отчёта с помощью функций и f-строк. |

## Итог

Проект демонстрирует базовые навыки работы с функциями в Python и может использоваться в портфолио тестировщика как пример практики по языку.
