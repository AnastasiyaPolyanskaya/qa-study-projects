# Формирование отчёта через f-строки (Python)

Учебный мини-проект, демонстрирующий создание многострочного текстового отчёта с использованием f-строк.  
Задача практикует:

- форматирование текста;
- подстановку переменных;
- расчёт итоговой суммы;
- работу с многострочными строками (`""" ... """`).

| Файл | Описание |
|------|----------|
| `task_5_string_format_report.py` | Формирует оформленный текстовый отчёт о покупке товара. |
