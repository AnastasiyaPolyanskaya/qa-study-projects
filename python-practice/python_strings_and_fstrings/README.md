# Python: Строки и f-строки

Учебный проект для практики работы со строками в Python  
и использования форматирования через **f-строки**.

Проект состоит из двух частей:

---

## 1. `strings_basic/` — базовые операции со строками

Файл: **task_strings_all.py**

Содержит примеры:

- вывод строки полностью  
- доступ к символам по индексу  
- срезы строк  
- получение чётных индексов  
- поиск предпоследнего символа  
- выделение последнего слова  
- удаление символов  
- работа с регистром  

Файл объединяет все короткие задания в один удобный учебный скрипт.

---

## 2. `f_strings/` — работа с f-строками

Файл: **task_fstrings_all.py**

Реализованы примеры:

- вывод имени через f-string  
- преобразование регистра в одной строке  
- вычисление возраста фильма  
- конвертация суммы из рублей в доллары и евро  

Все упражнения собраны в одном демонстрационном скрипте.

---

## Цель проекта

Показать базовые навыки:

- работы со строками  
- обработки пользовательского ввода  
- применения f-строк  
- написания простых учебных скриптов  

Проект подходит для портфолио начинающего тестировщика или Python-разработчика.
