# Функции и оператор return (Python)

Учебный мини-проект, посвящённый практике работы с функциями и оператором `return`.  
Задачи показывают, как выносить вычисления в отдельные функции, использовать результат в дальнейших расчётах и строить простые консольные мини-калькуляторы.

## Содержимое проекта

- функции суммы и среднего арифметического;
- расчёт скидки с использованием `return`;
- собственные реализации `max` и `min`;
- использование результата функции в дальнейших вычислениях;
- мини-калькулятор из четырёх операций.

---

## Файлы проекта

| Файл | Описание |
|------|----------|
| `task_1_sum_return.py` | Функция, возвращающая сумму трёх чисел. |
| `task_2_average_return.py` | Функция, считающая среднее арифметическое списка чисел. |
| `task_3_discount_return.py` | Функция для расчёта цены товара со скидкой. |
| `task_4_my_max_min.py` | Собственные реализации функций поиска максимума и минимума. |
| `task_5_calc_4_ops.py` | Мини-калькулятор из четырёх операций (сложение, вычитание, умножение, деление). |

---

Проект выполнен в рамках обучения Python и подготовки портфолио тестировщика.
