# Работа со строками (Python)

Учебный мини-проект, посвящённый практике работы со строками.  
Задачи демонстрируют навыки обработки текста, поиска символов, замены слов и разбиения строки.

## Содержимое проекта

- подсчёт длины строки, первый и последний символы;
- вывод строки в верхнем и нижнем регистре;
- разворот строки и замена пробелов на "-";
- вывод только букв из строки;
- подсчёт количества выбранного символа;
- поиск слова в тексте;
- замена слова в строке;
- разбиение строки на слова.

---

## Файлы проекта

| Файл | Описание |
|------|----------|
| `task_1_string_operations.py` | Базовые операции со строками: длина, регистр, разворот строки, замена пробелов. |
| `task_2_letters_only.py` | Вывод только букв из строки. |
| `task_3_count_symbol.py` | Подсчёт количества выбранного символа. |
| `task_4_find_word.py` | Поиск слова в тексте. |
| `task_5_replace_word.py` | Замена слова в строке. |
| `task_6_split_words.py` | Разбиение строки на отдельные слова. |
