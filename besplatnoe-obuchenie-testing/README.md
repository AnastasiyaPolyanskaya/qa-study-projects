# Бесплатное-обучение.рф — тестирование веб-форм

## Описание проекта
Учебный проект по исследовательскому и функциональному тестированию сайта «бесплатное-обучение.рф».  
Проверялись формы регистрации, каталог курсов, поведение UI, корректность валидации.

---

## Структура проекта
- test-cases — тест-кейсы
- checklists — чек-листы
- bug-reports — баг-репорты
- screenshots — скриншоты (при наличии)

---

## Что выполнено
- исследовательское тестирование сайта  
- тест-кейсы по основным сценариям  
- проверки валидации полей  
- сравнение поведения на ПК и мобильной версии  
- оформление баг-репортов  

---

## Цель работы
- функциональное тестирование  
- тест-дизайн  
- анализ дефектов  
- оформление тестовой документации
