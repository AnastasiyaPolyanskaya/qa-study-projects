# Скриншоты - бесплатное-обучение.рф

Раздел содержит визуальные материалы, использованные в процессе тестирования сайта «бесплатное-обучение.рф».

## Что содержится в этой папке

- Скриншоты ошибок, найденных во время тестирования  
- Подтверждения шагов воспроизведения дефектов  
- Скриншоты валидации полей  
- UI-несоответствия  
- Скриншоты поведения форм регистрации и авторизации  
- Материалы для подтверждения баг-репортов  
- Если проводилось тестирование через DevTools — сетевые логи, HAR-файлы (по желанию)

## Назначение раздела
Скриншоты помогают подтвердить найденные дефекты, визуализировать проблему и улучшить качество баг-репортов.
