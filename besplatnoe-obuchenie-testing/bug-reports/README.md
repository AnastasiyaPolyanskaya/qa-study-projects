# Баг-репорты — бесплатное-обучение.рф

Раздел содержит баг-репорты, найденные в ходе исследовательского и функционального тестирования сайта «бесплатное-обучение.рф».

## Что содержится в этой папке

- Ошибки валидации (регистрация, авторизация, формы обратной связи)  
- Функциональные дефекты поведения кнопок и форм  
- Некорректная работа UI-элементов  
- Несоответствия макетам и требованиям  
- Баги отображения контента (текст, изображения, верстка)  
- Ошибки в мобильной версии  
- Ссылки, ведущие не туда или не работающие  
- Дефекты, подтвержденные скриншотами  

## Структура баг-репортов
Каждый баг-репорт оформлен по стандарту:

- **ID**  
- **Название** (краткое описание проблемы)  
- **Окружение** (браузер, устройство, ОС)  
- **Шаги воспроизведения**  
- **Ожидаемый результат**  
- **Фактический результат**  
- **Приоритет / Серьезность**  
- **Прикрепленные материалы** (скриншоты, видео, HAR)  

## Назначение раздела
Баг-репорты фиксируют выявленные дефекты и помогают оценить стабильность и корректность работы сайта.
