# Чек-листы — бесплатное-обучение.рф

Раздел содержит чек-листы, составленные в рамках исследовательского и функционального тестирования сайта «бесплатное-обучение.рф».

## Содержание раздела

- Проверка формы регистрации  
- Проверка формы авторизации  
- Проверка отображения каталога курсов  
- Проверка функциональности карточек курсов  
- Проверка UI-элементов (кнопки, ссылки, хедер, футер)  
- Проверка адаптивности и мобильной версии  
- Проверка обработки ошибок  
- Проверка нестандартных и edge-case сценариев  
- Проверка корректности текстов и отображения контента  

## Назначение чек-листов

Чек-листы используются для быстрого покрытия критически важных функций сайта и позволяют:

- системно проверять бизнес-логику,
- выявлять UI-несоответствия,
- фиксировать ошибки отображения,
- контролировать основные пользовательские сценарии,
- ускорять регрессионное тестирование.

## Формат чек-листов

Каждый чек-лист оформлен в структурированном виде и включает:

- Название раздела  
- Список проверок  
- Отметки о прохождении  
- Дополнительные комментарии
