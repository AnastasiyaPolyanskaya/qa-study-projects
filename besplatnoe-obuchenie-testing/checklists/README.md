# Чек-листы — бесплатное-обучение.рф

Раздел содержит чек-листы, составленные в рамках тестирования платформы бесплатное-обучение.рф.

## Содержание
- Проверка UI-элементов
- Проверка функциональности кнопок
- Проверка валидации полей
- Проверка ошибок
- Проверка отображения курсов и уроков
- Проверка мобильной версии
- Нестандартные сценарии и edge-cases

## Описание
Чек-листы используются для быстрой оценки работоспособности интерфейса, логики и поведения системы при вводе данных, навигации и взаимодействии пользователя.
