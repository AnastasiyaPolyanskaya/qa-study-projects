# Pairwise-тестирование

Pairwise-тестирование — это техника тест-дизайна, которая уменьшает количество тестов, сохраняя необходимое покрытие. Метод основан на проверке всех возможных попарных сочетаний значений параметров.

## Когда применяется

Техника используется при тестировании объектов, где важно взаимодействие нескольких параметров:

- формы регистрации
- фильтры и настройки
- конфигурации товара
- варианты выбора (цвет, размер, тип доставки, способ оплаты)

## Пример

Есть параметры:

Параметр | Значения
--------|----------
Тип доставки | Курьер, Самовывоз, Почта
Способ оплаты | Картой, Наличными
Размер посылки | Маленькая, Средняя, Большая

Полное количество тестов: 3 × 2 × 3 = 18.

Pairwise позволяет сократить количество тестов до 4–6, сохранив покрытие взаимодействий.

## Пример pairwise-таблицы

№ | Тип доставки | Оплата | Размер
---|--------------|--------|--------
1 | Курьер | Картой | Маленькая
2 | Курьер | Наличными | Средняя
3 | Самовывоз | Картой | Средняя
4 | Самовывоз | Наличными | Большая
5 | Почта | Картой | Большая
6 | Почта | Наличными | Маленькая

## Преимущества техники

- уменьшение количества тест-кейсов
- проверка всех важных взаимодействий параметров
- оптимизация регрессионного тестирования
- снижение затрат времени и ресурсов

## Инструменты для генерации Pairwise

- Microsoft PICT
- AllPairs
- Pairwise Online Tool
- TestersPairwise
