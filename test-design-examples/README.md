# Примеры применения техник тест-дизайна

## Описание проекта
Учебный проект с примерами использования техник тест-дизайна:  
классы эквивалентности, граничные значения, таблицы решений.

---

## Структура проекта
- equivalence-classes — классы эквивалентности  
- boundary-values — граничные значения  
- decision-tables — таблицы решений  

---

## Что выполнено
- наборы тестов по каждой технике  
- визуальные таблицы  
- пояснение логики и выбора тестов  
- примеры на реальных формах
